# RealBlockPhysics
Lift and throw blocks just like what you want!

Images at:
https://www.spigotmc.org/resources/realblockphysics-⚡-the-real-way-block-should-be.63084/

The block-throwing plugin!
​
## Background
When I was a child, I played HarvestMoon and it is quite fun to grab a stuff, throw it anywhere. And so I thought it would be interesting if I implement it to minecraft.
